This folder is where locally-hosted case study images should go.

Right now, all case study pages link to images hosted on your old
Wix site (static.wixstatic.com). That works only as long as those
files stay live on Wix. Before you retire the old site, download
each image referenced in the case-*.html files and save it here,
then update the corresponding <img src="..."> in each HTML file to
point to "images/your-filename.jpg" instead of the wixstatic.com URL.
